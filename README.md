
# Streamlit Table-Based User Editor

## Features
- Editable table UI
- Add/remove rows
- Save to S3

## Setup
1. Add secrets.toml or Streamlit Cloud secrets:

```
AWS_ACCESS_KEY_ID = "..."
AWS_SECRET_ACCESS_KEY = "..."
BUCKET = "deepaa-algo-trade"
KEY = "web_data/user_details.json"
REGION = "ap-south-1"
APP_PASSWORD = "1234"
```
